Implementación de Máquina Virtual en Azure

1. Creación de la Máquina Virtual
  • Proveedor: Microsoft Azure
  • Nombre: evaluación1
  • Región: Canadá Central
  • Imagen: Ubuntu Server 24.04 LTS
  • Tamaño: Standard_B1ms (1 vCPU, 2 GiB RAM)
  • Puertos habilitados: 80 (HTTP), 443 (HTTPS), 22 (SSH)
2. Acceso a la Máquina Virtual

Se accedió a la máquina virtual mediante SSH utilizando el siguiente comando:

ssh -i evaluacion1_key.pem vichogonza@130.107.144.135
3. Actualización de Paquetes

Se realizó la actualización de los paquetes del sistema con el siguiente comando:

sudo apt update
4. Instalación y Verificación de Apache2
  • Instalación de Apache2: 
    sudo apt install apache2 -y
  • Verificación del estado del servicio: 
    sudo systemctl status apache2
5. Clonación del Repositorio y Evidencias

Desafortunadamente hubo un problema al realizar la clonación del repositorio que contenía la página web, eliminamos todo lo relacionado al repositorio y a la clonación (.git), utilizamos sudo rm -rf ./* y agregamos otro sudo rm -rf .git para eliminar el archivo .git

sudo rm -rf ./*
sudo rm -rf .git
6. Clonación final del repositorio

Finalmente se le hicieron modificaciones a los archivos html para que se ajusten a los requerimientos del proyecto, se utilizó el siguiente comando para clonar el repositorio:

git clone https://github.com/Sonic2014z/Eva1ComptuacionNube .
